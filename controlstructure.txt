Control structure:
    - Simple If condition
    - if-else condition
    - if else ladder stmt
    - nested if condition
simple if condition:
    if(condition stmt)//eg : if sony avalible then block stmt execute
    {
        //block stmt
        purchase the tv
    }
if else condition:
    if(condition stmt){
        //block stmt
    }else{
        //block stmt
    }
if else ladder:
if(condition stmt 1){//if sony avalible
    //block stmt
    purchase
}else if(condition stmt 2){//if samsang avalible
    //block stmt
    purchase
}else{ //if not then do the else block 
    //block stmt
    come home
}
Nested if:
    Parent condition and child conditions
if(condition stmt){//if condition true then go to the block
    //block stmt
    if(condition stmt){
        //block stmt
    }
}
